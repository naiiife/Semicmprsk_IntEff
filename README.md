# Semicmprsk_IntEff
Randomized interventional effects in semicompeting risks

## generatedata.R
To generate semi-competing risks data with a time-varying confounder

## fit_p.R
Functions to estimate interventional effects

## simulation1.R
Simulation Setting 1: no frailty (parallel computation required, job file run1.sh)

## simulation2.R
Simulation Setting 2: normal frailty (parallel computation required, job file run2.sh)

## simulation3.R
Simulation Setting 3: gamma frailty (parallel computation required, job file run3.sh)

## simulation_results.zip
Simulation results, including 3 folders corresponding to 3 simulation settings

## analysis.R
To summarize simulation results, to plot estimated cumulative incidence functions, and to calculate bias
